# Saurabh Khule - Personal Portfolio

A modern, responsive personal portfolio website built with React.js, Tailwind CSS, and Framer Motion.

## Features

- **Modern Design**: Glassmorphism effects, smooth animations, and clean typography.
- **Responsive**: Fully responsive layout for mobile, tablet, and desktop.
- **Dynamic Content**: All data is separated in `src/data/resume.js` for easy updates.
- **Interactive**: Smooth scrolling, hover effects, and contact form UI.

## Project Structure

```
src/
├── assets/         # Images and static assets
├── components/     # Reusable UI components (Navbar, Hero, etc.)
├── data/           # Resume data source (resume.js)
├── pages/          # Page layouts (if needed in future)
├── App.jsx         # Main application component
├── main.jsx        # Entry point
└── index.css       # Global styles and Tailwind directives
```

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```
